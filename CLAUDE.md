# Landing Page — Экскурсовод

## Проект

Лендинг для экскурсовода по Нижнему Новгороду. Сайт должен быть максимально быстрым при загрузке, но с возможностью получать динамические данные с бэкенда (карточки экскурсий и т.д.) при масштабировании.

## Стек

- **Сборка:** Vite
- **Фреймворк:** React 19 + TypeScript
- **Стили:** CSS Modules (без тяжёлых UI-библиотек — ради скорости)
- **Роутинг:** пока не нужен (одностраничный лендинг)
- **Данные:** на первом этапе — статические, далее — fetch с бэкенда

## Дизайн-референс

Сайт: https://spbmania.ru/

### Визуальный стиль

- Минималистичный, современный
- Цвета: белый фон, тёмный текст (#111827), акцентный зелёный (#396053)
- Кнопки: тёмный фон (#111827), белый текст, плавные hover-переходы
- Шрифты: sans-serif (Inter или аналог)
- Карточки с мягкими тенями (box-shadow)
- Адаптивный дизайн (mobile-first)

### Структура секций

1. **Header** — логотип, навигация (Экскурсии, Услуги, Отзывы, Оплата), телефон, иконка Telegram
   - **Телефон** — кликабельный, оформлен как `<a href="tel:+7...">`, чтобы на мобильных сразу открывался набор номера
   - **Иконка Telegram** — ведёт на `https://t.me/<username>`, открывается в новой вкладке (`target="_blank"`, `rel="noopener noreferrer"`), с осмысленным `aria-label` («Написать в Telegram»)
   - Набор мессенджеров может расшириться позже (Max, Viber и т.п.) — добавлять через конфиг
   - Контакты (номер, username'ы) вынесены в конфиг `src/config/contacts.ts`, а не захардкожены в разметке
2. **Hero** — полноэкранный баннер с фоновым изображением, заголовок, подзаголовок, CTA-кнопка
3. **О гиде** — краткое описание, опыт, преимущества (иконки + текст)
4. **Экскурсии** — карточки экскурсий (фото, название, цена, краткое описание) — в будущем данные с бэка
5. **Галерея** — фото с экскурсий (сетка 3 колонки)
6. **Отзывы** — карточки отзывов клиентов
7. **CTA-форма** — форма заявки (имя, телефон, сообщение)
8. **Footer** — контакты, ссылки, копирайт

## Команды

```bash
npm install        # Установка зависимостей
npm run dev        # Dev-сервер (http://localhost:5173)
npm run build      # Продакшн-сборка
npm run preview    # Превью продакшн-сборки
```

## Принципы

- Никаких тяжёлых библиотек без явной необходимости
- Изображения — WebP/AVIF с lazy loading
- Компоненты — маленькие, переиспользуемые
- CSS — без !important, без глобальных стилей (кроме reset)
- Семантический HTML (header, main, section, footer)
- Доступность: alt для изображений, aria-label для интерактивных элементов
