Условия и порядок оплаты 

Для физлиц:
- Для физлиц бронирование экскурсии по авансу 1000 руб. После подтверждения бронирования экскурсии высылаем QR-код для внесения аванса. Чек предоставляется. Окончательный расчет в день экскурсии. 

Для организаций:
- Услуги для организаций оказываются по полной предоплате. Заключается договор оказания услуг с самозанятым, для выставления счета потребуется ИНН организации.
